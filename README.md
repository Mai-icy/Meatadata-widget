# MusicTager

一个带有图形界面的mp3 flac元数据补全工具

## 简介 Introduction

这是基于Python3以及Pyqt5的图形界面工具，利用网易云api的数据，补全歌曲元数据的工具。

## 特性 Features

- 拥有方便操作的用户图形界面
- 使用了网易云，酷狗歌曲数据
- 可以手动编辑元数据，自定义上传专辑图片
- 可以补全mp3，flac, m4a文件
- 可以下载歌词
- 可以自动补全元数据

## 运行环境与依赖 Running environment and requirements

- Python 3
- Pyqt5 module
- mutagen module
- requests module
